# Satellite Inventory Analyzer (Physical Proof)

This skill focuses on 'Ground Truth' by analyzing public or premium satellite imagery to detect shadow lengths in floating-roof oil tanks or stockpile volumes.

## Core Mechanics
1. Scrape or query satellite imagery for key hubs (Cushing, Singapore, China Strategic Reserves).
2. Calculate the height of floating lids or volume of coal/iron ore piles using shadow geometry.
3. Compare real-world volume with reported inventory data to find 'Institutional Lies'.
4. Signal a trade setup if real supply is significantly higher/lower than reported.

# Social Sentiment Bait Agent (Deception Layer)

This skill uses autonomous 'Clone Agents' to interact on social forums (Reddit, X, Discord) to gauge where the crowd is positioning and where they are likely to panic.

## Core Mechanics
1. Deploy 3-5 agents to ask strategic questions (e.g., 'Is anyone else longing Gold at 2400?').
2. Analyze responses to identify clustered stop-loss zones and emotional pain points.
3. Map the 'Retail Liquidity' based on actual human responses, not just chart patterns.
4. Guide bionicbot to enter trades exactly when these retail clusters are about to be 'swept' by whales.

## Examples
- "Deploy a sentiment probe in the 'WallStreetBets' discord: find out the consensus stop-loss for the current Oil trade."
